Privacy Policy

This Privacy Policy explains how the Real-Time Spam Mail Detection application collects, uses, and protects user data.

Information We Access

Our application uses the Google Gmail API to read email message content for the purpose of detecting spam messages. We only request the following permission:

  • Gmail Read-Only Access

The application does not modify, send, or delete any emails.

How We Use Your Data

Email data accessed through Gmail is used only for spam detection using machine learning models. The content is analyzed temporarily and is not permanently stored or shared with third parties.

Data Storage

We store minimal information required to run the service, such as prediction results (spam or not spam). Sensitive email content is not permanently stored.

Third-Party Services

This application uses Google APIs and services. Use of these services is subject to Google's Privacy Policy.

Security

We take reasonable measures to protect user information and ensure that data access is limited to the functionality of the application.

Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page.
